# House Price Prediction (A city in USA as an Example) - Simple Linear Models

This project demonstrates a simple ML pipeline using housing dataset:
- Data loading and train/test split
- Feature scaling
- Models: Linear Regression, Ridge, Polynomial (optional)
- Evaluation (RMSE, R2)
- Save best model and scaler with joblib
- Simple predict function using saved model

## How to run
1. Create venv and install requirements:

2. Run `house_price_prediction.py` in PyCharm or `python house_price_prediction.py`.

## Files
- `house_price_prediction.py` : main script
- `saved_models/` : (generated) contains the best model and scaler
